pre-scientific work in physics LaTeX

I'm currently occupied in writing and finishing a work called "Fach­bere­ich­sar­beit" for my grammar school graduation. My thesis has to do with current de­vel­op­ments and theses in the debate of the ac­cel­er­at­ing expansion of the universe and theories dealing with these topics. It gives also an overview on the history of all the ob­ser­va­tions and dis­cus­sions that led us to the state of play.

Due to this important long-term task, it's impossible for me to engage in other projects till the release.

Particle Zoo SVG

This is my try to visualize the particles of the physical standard-model, showing not only the names and groups, but also some data like mass and spin. I created it as part of my pre-scientific work mentioned above.

Litotex PHP

Litotex is a team work of the Litotex community of which I am part of. It's a PHP framework that has to be developped so far, so that the code is readable and the per­for­mance acceptable. The concepts of DRY and mod­uliza­tion are the aim of that project. And of course: It's meant to make fun de­vel­op­ping (on) it.

Unity-Opera Python

Unity-Opera is a small Python-Script that adds Unity in­te­gra­tion to the Opera Webbrowser. Unity is the new Desktop en­vi­ron­ment of the very popular Linux dis­tri­b­u­tion Ubuntu.

Since I'm using Gnome2/3 instead of Unity I stopped working on it, but nev­er­the­less the script behaves well.

Jamendo-Downloader Bash

This small Shell-Script can help you in enlarging your music repertoire! Just pass a tag (e.g. your favorite genre), the album or another list of music tracks or albums to the script and let them download in OGG or MP3. It uses the official API of Jamendo.

Batch-test your programs Bash

This script executes one given program with a bunch of input files and compares the output with the solution-files (if given). Very usefull when training for the Olymopiad in In­for­mat­ics, which provides huge testdata for every problem.
